# Sales Data Analysis System

## Overview
The **Sales Data Analysis System** is a data analytics project designed to analyze and visualize sales data to extract meaningful business insights. The system processes structured sales data, stores it in a database, performs analytical queries, and generates visualizations to understand revenue trends, category performance, and overall sales behavior.

This project demonstrates end-to-end data analysis using **Python, R, SQL, and SQLite**, making it suitable for academic projects and data analytics portfolios.

## Objectives
- Analyze sales data to identify revenue patterns
- Determine top-performing product categories
- Perform monthly and category-wise sales analysis
- Generate statistical summaries for decision-making
- Visualize sales trends using charts and graphs

## Technologies Used
- **Python**: Pandas, Matplotlib
- **R**: ggplot2, dplyr
- **SQLite3**: Database storage
- **SQL**: Data querying and aggregation
- **CSV**: Data input format
